    Chapter 11：类和对象初步#

    面向对象的程序设计：”抽象“，“封装”，“继承”，“多态”等特点。

    类的定义和使用：#

    class ClassName:
{
    public:
    variable X;
    function Y;
    private:
    ...
    protected:
    ...
};

    “访问范围说明符“有 public，private，protected。

    访问对象的成员#

    可以使用“对象名.成员名“，也可以使用”指针->成员名“的方式来访问

    类还有另外一种写法，将 class 关键字换成 struct 关键字。

    struct CRectangle
{
    public:
    int w, h;
    void Init(int w_, int h_);
}

    没有成员函数的 struct 还是结构，结构变量不是对象；有成员函数的 struct 就是类。写成 struct 的类和写成 class 的类只在类成员的可访问范围方面有一点小差异。

    类成员的可访问范围#

    1. private：用来指定私有成员，只能被该类的成员函数内部才能被访问。
    2. public：用来指定公有成员，任何地方都可以访问
    3. protected：用来指定保护成员，在继承中会被使用

    如果不使用可访问范围符则默认是 private

    内联成员函数#

    class MyClass:
{
    inlinee void func();
}

    内联成员函数执行时间较快，但是会增大可执行文件的体积。如果执行时间较长的内联函数，只会额外增加开销。

    Chapter 12：类和对象进阶#

    构造函数#

    c++的变量初始化：全局变量默认为0；局部变量是动态分配在栈上，是随机的。为什么不将所有局部变量都初始化为0:因为一个函数的局部变量在内存中的地址，每次函数被调用时都可能不同，所以自动初始化的工作就不是一次性的了；

    构造函数 constructor，在创建一个类的时候会调用的函数。一个类可以有多个构造函数。如果类的设计者没有编写构造函数，那么编译器就会自动生成一个没有参数的构造函数。无参数构造函数被称为默认构造函数 default constructor。注意：构造函数并不是负责为对象分配内存空间，而是初始化内存空间。构造函数可以有多个，编译到能生成对象的语句时，编译器会根据参数决定调用哪个构造函数。如果没有任何参数，则是无参构造函数。

    复制构造函数#

    复制构造函数是构造函数的一种，也称为拷贝构造函数，它只有一个参数，参数类型是本类的引用。如果类的设计者没有编写复制构造函数，编译器就会自动生成复制构造函数。注意：默认构造函数（无参构造函数）不一定存在，但是复制构造函数总是会存在。

    class MyClass:
{
    MyClass(const MyClass& c){...} //正确
    MyClass(MyClass c){...} // 错误
}

    复制构造函数会被调用的三种情况：

    1. 用一个对象去初始化同类的另一个对象时，会引发复制构造函数被调用
    2. 如果函数F的参数是类 A 的对象，那么函数F被调用时，类 A 的复制构造函数将被调用。
    class A
{
public:
    A()
    {
        cout << "Default construct called" << endl;
    };
    A(const A &a)
    {
        cout << "Copy construct called" << endl;
    }
};

void Func(A a) 
{
    // void Func(A &a)
    // void Func(const A &a)
    return;
}

int main()
{
    A a;
    Func(a);
    return 0;
}

    如果运行以上程序，得到的输出为：

    Default construct called
Copy construct called

    如果在函数参数中加一个引用，则会把 a 直接传进函数 Func 中，不会调用复制构造函数：

    Default construct called

    但是注意，如果传参数是引用，函数被调用的时候，型参被改变，实参的值也会被改变，这时候就要加上 const 关键字了。

    1. 函数的返回值是类的对象，则函数返回时，类的复制构造函数被调用

    析构函数#

    析构函数是成员函数的一种，名字与类名相同，但前面要加上~，没有参数和返回值。一个类有且仅有一个析构函数。析构函数在对象消亡时被调用。
